# Get Dataset
Run the following command to load dataset:
```
py scrapper.py
```

Note: you must have the following installed in your python environment:
```

certifi==2019.11.28
chardet==3.0.4
html5lib==1.0.1
idna==2.8
lxml==4.4.2
requests==2.22.0
six==1.14.0
soupsieve==1.9.5
urllib3==1.25.7
webencodings==0.5.1
```
